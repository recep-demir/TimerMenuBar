# PrayerTimer for macOS

**PrayerTimer** is a lightweight, native macOS menu bar application designed to help you track Islamic prayer times seamlessly throughout your day. Built with SwiftUI, it provides real-time countdowns, location-based prayer timings, and notification reminders.

## üöÄ Key Features

* **Menu Bar Countdown:** See the remaining time to the next prayer directly in your menu bar.
* **Dynamic Icons:** The icon changes based on the current time (e.g., Crescent for Isha, Sun for Dhuhr).
* **City Search:** Change your location easily by searching for any city worldwide.
* **Dual Calendar:** Displays both Gregorian and Hijri dates in the menu.
* **Notifications:** Get notified before the prayer time (35-minute pre-reminder) and exactly when it's time to pray.
* **Native Design:** Minimalist UI that integrates perfectly with macOS Light and Dark modes.
